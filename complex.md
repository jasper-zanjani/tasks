# Complex challenges
- Assign a range of values to an array (`arr-rng`)
- Zip up elements of two arrays (`arr-zip`)
- Read a file into an array, then print it out (`cat`:`arr-fi-out`) (YUG: 621)
- Loop over an array and write the values to a file (`loop-arr-fo`)
- Loop over an array and write filenames to separate files (`loop-arr-fo`)
- Fibonacci sequence
- Validated input and display of string (`inp-out-cf`): display inputted information with validation (YUG:615)
  - Ancestors: `inp-out`
- Simple input and output (`inp-out`): display inputted information (YUG:615)
  - Descendants: `inp-out-cf`, `inp-out-mth`
- Display multiple iterations of string (`out-loop`)
  - Ancestors: `out`
- Convert temperature (`temp`: `inp-out-mth`) (YUG:614)
  - Ancestors: `inp-out`; `mth`
- Determine whether inputted year is a leap year (`inp-out`,`mth`) (YUG: 622)
- String concatenation (`str-cat`)
- Search for a string in a file and print it (`grep`) (YUG: 618)
  - Ancestors: `re`;`fi`;`out`
- Indent (`fi-loop-fo`): insert fixed indentation to all lines of a text file
- Display last line of a text file (`fi-loop-out`): display last line of a text file
- Convert an IP address to binary (`iptobin`: `inp-mth-loop-out`) (YUG:626)
